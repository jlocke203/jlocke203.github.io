Blog

A New Way to Talk to My Computer | September 20, 2020

This was my first blog attending Flatiron. I explain how at first I didnt know anything about programming and how I continued to work hard and was able to complete my first ever project. Dedication can take you a long way especially when it comes to code. In this blog I emphasize that alot.

The Art of Programming | October 5, 2020

In this blog I talk about the average life of a software engineer. I get into the good things about learning to code and the bad. Also I talk about how it defers from other jobs and what changes you are gonna have to make in your life to be the the best develeoper you can be. My(Promise => To You) | October 20, 2020

The History of React | November 2, 2020

In this blog I go a little back into history and talk about React in the recent years and what it has done and how it became the powerful tool that it is. I talk about the creator of react and the year that happend all in a timeline version.

Functional vs. Class Components. | November 17, 2020

Work

ArtWalk | Module 2

ArtWalk was made by myself and a fellow classmate Ben Looper. It is a CLI based application with the usage of Ruby on Rails, CSS, and Bootstrap. It was made for our second project at Flatiron School, ArtWalk is a social platform for artist to show off their art to other artists.

Holiday Shopping List | Module 4

This application was developed by myself and a fellow classmate named Michael Archer. Holiday shopping list is used for the holidays to keep track of all your gifts. It was created with React and CSS as a mini project of the fourth module at Flatiron School.

Larrys Lemon Lot | Module 4

Larrys Lemon Lot is an application used with React in the frontend and Ruby on Rails in the backend also CSS and Bootstrap. It was developed by myself and a fellow classmate named Payton Gray for our Module 4 project at Flatiron School.
